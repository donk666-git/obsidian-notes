- **第一步：启动区块链网络和合约**
    
    - 打开**第一个终端**。
        
    - 进入 `test-network` 目录并启动网络、创建通道、部署合约。
        
        
        
        ```
        cd ~/hyperledger/fabric/scripts/fabric-samples/test-network
        ./network.sh down
        ./network.sh up createChannel
        ./network.sh deployCC -ccn certichain -ccp ../chaincode/certichain -ccl go
        ```
        
    - 启动成功后，**保持这个终端窗口不要关闭**。
        
- **第二步：现在才启动后端**
    
    - 打开**第二个新的终端**。
        
    - 进入你的后端项目目录。
        
        Bash
        
        ```
        cd ~/certichain-api
        ```
        
    - 运行你的 Go 程序。
        
        Bash
        
        ```
        go run main.go
        ```
