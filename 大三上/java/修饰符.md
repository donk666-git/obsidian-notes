#### ✅ **情况：子类与基类不在同一包**

- 子类**可以继承并使用基类的 `protected` 成员**，但是：
    
    - 只能通过 **自身继承下来的成员**来访问；
        
    - 不能通过“基类对象”来访问。

#### 访问控制与继承

- 父类中声明为 public 的方法在子类中也必须为 public。
    
- 父类中声明为 protected 的方法在子类中要么声明为 protected，要么声明为 public，不能声明为 private。
    
- 父类中声明为 private 的方法，不能够被子类继承。
