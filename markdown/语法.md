## 2️⃣ 段落与换行

- 段落直接换行即可
    
- 强制换行：在行尾加两个空格
    

`这是第一行   这是第二行`

## 3️⃣ 字体样式

`*斜体* 或 _斜体_ **加粗** 或 __加粗__ ***加粗斜体*** 或 ___加粗斜体___ ~~删除线~~`

## 4️⃣ 列表

### 无序列表

`- 项目一 - 项目二  - 子项目 * 或 + 也可以`

### 有序列表

`1. 第一项 2. 第二项   1. 子项   2. 子项`
